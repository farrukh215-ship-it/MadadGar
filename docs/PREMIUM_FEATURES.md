# Premium Features (WhatsApp-style)

These 5 premium features are available to users with an active Premium subscription:

1. **Delete message for everyone** – Remove your sent messages from both sides of the chat (free: delete for me only)
2. **Larger media sharing** – Send videos up to 100MB (free: 25MB limit)
3. **Message pinning** – Pin important messages in a chat (DB ready, UI TBD)
4. **Chat themes** – Custom backgrounds per chat (DB ready, UI TBD)
5. **Precise last seen** – Show exact "last seen at X:XX" for contacts (free: basic online/offline)

## Existing Premium Benefits

- Unlimited interests (free: 5)
- Top visibility in interest lists
- Premium badge on profile in chat discovery
